This website uses **Pirsch Analytics** for aggregated traffic insights.

What is collected:
- page-level visit metrics;
- aggregated country-level location statistics.

What is not stored in this repository:
- raw IP addresses;
- personally identifying visitor profiles.

A lightweight Cloudflare Worker is used only to fetch and expose aggregated analytics data for the footer stats module.
